### 扩展注释高亮配置
打开 vscode 的 settings.json 文件
添加一种高亮注释（也可以直接改颜色）

```js
"better-comments.tags":[
  {
    "tag":"1",
    "color":"#84a9ac",
    "strikethrough":"false",
    "backgroundColor":"transparent"
  }
]
// 使用
// 1 新的注释
```
